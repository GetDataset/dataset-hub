Quick Start

This section shows how to load your first dataset using the unified .get_<dataset>() API.

Basic example

Load any available dataset in one line:

import dataset_hub

titanic = dataset_hub.classification.get_titanic()
print(type(titanic))  # pandas.DataFrame
